# ⚡ 快速配置指南

## 🎯 5 分钟完成真机调试配置

### 第 1 步：查看本机 IP（1 分钟）

**Windows 用户**：
1. 按 `Win + R`，输入 `cmd`，回车
2. 输入 `ipconfig`，回车
3. 找到 "IPv4 地址"，例如：`192.168.1.100`

**Mac 用户**：
1. 打开"系统偏好设置" → "网络"
2. 查看当前连接的 WiFi，记下 IP 地址
3. 或者打开终端，输入：`ifconfig | grep "inet "`

### 第 2 步：修改配置文件（2 分钟）

1. 打开文件：`mp-weixin/config.env.js`

2. 找到第 60 行左右：
```javascript
debug: {
  baseURL: 'http://YOUR_LOCAL_IP:8080/nodejsn73cv/',
  ...
}
```

3. 将 `YOUR_LOCAL_IP` 替换为你的 IP，例如：
```javascript
debug: {
  baseURL: 'http://192.168.1.100:8080/nodejsn73cv/',
  ...
}
```

4. 保存文件（`Ctrl + S` 或 `Cmd + S`）

### 第 3 步：启动后端服务（1 分钟）

确保后端服务已启动，监听 `8080` 端口。

### 第 4 步：真机调试（1 分钟）

1. 在微信开发者工具中点击"真机调试"
2. 用手机微信扫码
3. 完成！

## 🚨 常见错误

### 错误 1：手机显示"网络请求失败"

**原因**：手机和电脑不在同一 WiFi

**解决**：确保手机和电脑连接同一个 WiFi 网络

---

### 错误 2：控制台显示"YOUR_LOCAL_IP"

**原因**：忘记修改配置文件

**解决**：按照第 2 步重新修改 `config.env.js`

---

### 错误 3：IP 地址经常变化

**原因**：路由器使用 DHCP 动态分配

**解决**：在路由器中为电脑设置静态 IP（推荐）
