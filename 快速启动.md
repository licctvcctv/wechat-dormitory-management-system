# ⚡ 快速启动指南

## 🎯 3 步启动项目

### 步骤 1：导入数据库

```
1. 打开 MySQL 数据库
2. 创建数据库：nodejsn73cv
3. 导入文件：nodejsn73cv/db/nodejsn73cv.sql
```

### 步骤 2：启动后端（HTTPS）

**方法 A：使用 Node.js 脚本（推荐，不会闪退）**
```bash
cd nodejsn73cv
node start-https.js
```

**方法 B：双击 bat 文件（Windows）**
```
双击运行 nodejsn73cv/一键启动HTTPS.bat
```

**方法 C：使用 npm 脚本**
```bash
cd nodejsn73cv
npm run https
```

**macOS/Linux 用户**：
```bash
cd nodejsn73cv
npm install
npm run setup
npm run start:https
```

### 步骤 3：打开小程序

```
1. 打开微信开发者工具
2. 导入项目：选择 mp-weixin 目录
3. 详情 -> 本地设置 -> 勾选"不校验合法域名"
4. 真机调试（手机和电脑同一 WiFi）
```

## ⚠️ 注意事项

1. **数据库配置**: 修改 `nodejsn73cv/src/config.json` 中的数据库密码
2. **真机调试**: 必须使用 HTTPS 模式
3. **同一网络**: 手机和电脑需连接同一 WiFi
4. **防火墙**: 确保端口 8443 未被阻止

## 🆘 遇到问题？

### 常见问题速查

**问题**: 双击 bat 文件闪退
**解决**: 使用 Node.js 脚本代替
```bash
cd nodejsn73cv
node start-https.js
```

**问题**: 端口被占用
**解决**: Node.js 脚本会自动清理端口，直接运行即可

**问题**: npm 不是内部或外部命令
**解决**: 重新安装 Node.js，勾选 "Add to PATH"

**问题**: 真机调试图片无法加载
**解决**:
1. 使用 HTTPS 模式启动
2. 勾选"不校验合法域名"
3. 检查手机和电脑是否同一 WiFi

**问题**: 数据库连接失败
**解决**: 检查 `nodejsn73cv/src/config.json` 中的配置
